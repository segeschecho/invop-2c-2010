\section{Trabajo a futuro}

Una de las cosas que queda pendiente mejorar, es la complejidad de las heurísticas iniciales, ya que pueden ser mejoradas en ese aspecto. Además quedaron algunas ideas para mejorar la eficiencia de las heurísticas, en las que se destacan, la exploración de mas de un nodo inicial o recorrer el grafo de forma diferente.

Con respecto a las heurísticas de separación, es cierto que se puede mejorar también la complejidad de cada una ya que si bien logran un buen promedio de cortes, muchas veces no mejoran demasiado el tiempo de ejecución, y además se puede mejorar la eficiencia o realizar otros tipos de heurísticas, aunque esto último requiera mas tiempo para analizar el problema.

Queda pendiente continuar con las pruebas de configuraciones de los algoritmos para buscar la máxima performance del problema.

El modelo, para realizar este trabajo, no fué modificado del modelo que se obtuvo al al comienzo del trabajo. Quedaría entonces ver que resultados se pueden obtener cambiando el mismo, ya sea, agregando restricciones para forzar el orden de los colores o para describir alguna otra cualidad del problema, cualidades que no se analizaron a fondo por el tiempo y los requerimientos del trabajo. Agregando nuevas restricciones al problema, puede pasar además que se puedan hallar otras formas de realizar cortes, que puedan mejorar aún mas la performance.

Queda bastante claro entonces, que se pueden hacer y mejorar muchas cosas y que el problema de coloreo, visto como un problema de optimización combinatoria, esta bastante abierto en cuanto a nuevas ideas para resolver de manera mas eficiente el problema.
